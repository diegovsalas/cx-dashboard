¿Cómo funciona el reporte de incidencias?

1

Contacto

El cliente escribe "Hola" al +52 449 156 1238

→
2

Identificación

El bot solicita nombre, empresa y correo para registrar al cliente

→
3

Clasificación

Selecciona servicio, tipo de incidencia y categoría desde un menú interactivo

→
4

Descripción

El cliente describe el problema. El bot genera un folio automático

→
5

Seguimiento

El ticket se asigna a un agente. El cliente puede consultar su estatus en cualquier momento

Funciones disponibles para el cliente por WhatsApp

Crear Ticket

Escribe "Hola" al +52 449 156 1238 para iniciar el flujo guiado.

Consultar Ticket

Escribe "Consultar" o su número de folio (ej: TKT-0001) para ver el estatus.

Hablar con Agente

Escribe "Agente" para solicitar atención de una persona real.

Enviar Evidencia

Envía fotos o documentos que se vinculan automáticamente al ticket activo.
